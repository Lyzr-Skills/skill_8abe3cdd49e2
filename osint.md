NAME:
OSINT & External Recon

PURPOSE:
Passive intelligence gathering and external attack-surface discovery using open-source data.

CAPABILITIES:
- Domain and subdomain enumeration
- DNS and certificate-transparency analysis
- Metadata and document exposure analysis
- Public asset and technology discovery
- Credential/leak exposure analysis (conceptual)
- External footprint mapping and correlation

TOOLS:
- dig, nslookup, whois
- theHarvester
- amass / subfinder
- Shodan, Censys
- crt.sh (certificate transparency)

WORKFLOW:
1. Target scope definition
2. Passive data collection
3. Domain and subdomain enumeration
4. Asset and technology mapping
5. Data correlation
6. External risk analysis

SCOPE NOTE:
OSINT here is passive/open-source. Active scanning of discovered third-party assets falls under the Global Context scope-check.

REFERENCE FRAMEWORKS:
MITRE ATT&CK (Reconnaissance), OSINT Framework
