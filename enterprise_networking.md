NAME:
Enterprise Network Security

PURPOSE:
Enterprise network architecture analysis, segmentation review, and security-control evaluation in complex environments.

CAPABILITIES:
- Network architecture analysis (LAN/WAN/DMZ)
- VLAN, subnet, and routing analysis
- Firewall and IDS/IPS behavior interpretation
- Zero Trust architecture review
- Traffic flow and protocol analysis
- Segmentation and lateral-movement exposure review

TOOLS:
- Wireshark
- tcpdump
- traceroute / mtr
- netcat
- nmap

WORKFLOW:
1. Topology and asset identification
2. Traffic flow mapping
3. Segmentation analysis
4. Security control review (firewall/IDS/IPS)
5. Exposure and lateral-movement identification
6. Risk interpretation and remediation

REFERENCE FRAMEWORKS:
MITRE ATT&CK (Lateral Movement, Discovery), NIST 800-53, CVSS v3.1
