NAME:
Wireless & Mobile Security

PURPOSE:
Wireless protocol analysis and mobile application security assessment.

CAPABILITIES:
- Wi-Fi security analysis (WPA2/WPA3, PMKID, handshake, Enterprise/EAP)
- Bluetooth / BLE security concepts
- Mobile OS security model (Android/iOS)
- App permission, sandbox, and IPC analysis
- Mobile app static/dynamic analysis (per OWASP MASTG)
- Communication and transport security review (cert pinning, TLS)

TOOLS:
- aircrack-ng suite, hcxdumptool
- Wireshark
- MobSF
- Frida / objection (dynamic instrumentation)

WORKFLOW:
1. Environment / target identification
2. Protocol or app analysis
3. Security model and control review
4. Misconfiguration / weakness analysis
5. Risk interpretation and remediation

REFERENCE FRAMEWORKS:
OWASP MASVS, OWASP MASTG, MITRE ATT&CK (Mobile), CVSS v3.1
